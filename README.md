# bioinformatics-hemoglobin-analysis
# Comparative Analysis of Hemoglobin Protein Across Species

## Overview
This project analyzes the similarity of hemoglobin protein sequences across different species using bioinformatics tools.

## Species Studied
- Homo sapiens (Human)
- Mus musculus (Mouse)
- Bos taurus (Cow)
- Gallus gallus (Chicken)

## Tools Used
- NCBI Database
- BLAST (Basic Local Alignment Search Tool)
- Clustal Omega for sequence alignment
- Excel for visualization

## Methodology
1. Protein sequences were retrieved from the NCBI protein database.
2. Sequence similarity analysis was performed using BLAST.
3. Multiple sequence alignment was conducted using Clustal Omega.
4. Similarity percentages were recorded and visualized.

## Results
The analysis shows that hemoglobin proteins are highly conserved among mammals and show slightly lower similarity in birds.

Example similarity results:

| Comparison | Identity |
|-----------|----------|
| Human vs Mouse | ~85% |
| Human vs Cow | ~88% |
| Human vs Chicken | ~70% |
